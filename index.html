Island Health Explorer

Welcome to the Island Health Explorer. Here we provide a set of tools, powered by Google Earth Engine which provides an open-source platform to explore different GeoPhysical parameters across the incredible atolls.

How to use the Apps

Each app is provided with self explanatory instructions, with focus on simplicity and ease of use. Where ever a chart is produced the chart can be easily exported using the functionally already included within Google Earth Engine. However, users should be vary of using suitable time spans which can cause the apps to crash due to overloading of data. Currently the apps are categorized as below :

  1. Atoll Climate Explorer
  2. Atoll Geomorphology Explorer
  3. Atoll Weather Explorer
  4. Atoll Ocean Explorer
The whole platform is built open source with all of the code availble. Contribution and feedback is welcome and encouraged.
